# Calculadora Básica Multilingüe 🚀

Este proyecto corresponde a la práctica unificada de la **Unidad 1**, diseñada para demostrar la asimilación completa de los fundamentos de programación estructurada mediante la implementación de una calculadora aritmética básica en cuatro lenguajes diferentes de manera simultánea.

---

## 🛠️ Características del Proyecto

Para cumplir con los requisitos fundamentales de la unidad, cada uno de los programas incluye:
* **Definición precisa de variables:** Uso correcto de tipos de datos para admitir valores decimales continuos.
* **Estructuras de control condicionales:** Implementadas para discriminar la operación aritmética deseada y actuar como escudo ante errores críticos, previniendo explícitamente la división por cero.
* **Bucles continuos:** Ciclos integrados para mantener el flujo activo de la aplicación, evitando reinicios abruptos tras cada cálculo.
* **Delegación de tareas:** Modularización completa mediante funciones independientes para las operaciones de suma, resta, multiplicación y división.

---

## 📂 Tecnologías e Implementación

El repositorio contiene cuatro versiones independientes construidas bajo la misma lógica operativa, adaptadas a la sintaxis nativa de cada tecnología:

1. 🐍 **Python** (`calculadora.py`): Enfoque dinámico y legible.
2. 🔵 **C++** (`calculadora.cpp`): Control estricto de tipos de datos, uso de flujos `cin`/`cout` y bloques delimitados por llaves.
3. ☕ **Java** (`Calculadora.java`): Programación orientada a objetos pura, encapsulada en una clase homónima y uso de la utilidad `Scanner`.
4. 📊 **R** (`calculadora.R`): Enfoque analítico utilizando asignación por flecha (`<-`) y captura por consola estadística.
